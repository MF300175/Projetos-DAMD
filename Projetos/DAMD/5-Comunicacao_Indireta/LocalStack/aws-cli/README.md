## AWS CLI - LocalStack

Este diretório conterá arquivos de configuração e scripts de conveniência para
usar a AWS CLI apontando para o LocalStack.

Exemplos de scripts planejados:

- `criar-bucket-shopping-images.ps1`
- `listar-buckets.ps1`
- `listar-objetos-shopping-images.ps1`

Os arquivos `config` e `credentials` devem ser ajustados localmente conforme o
ambiente da máquina.
